# Bookshelf API
Sebagai tugas/proyek akhir dari kursus "Back-End Pemula Javascript" dari Dicoding, dengan membuat beberapa fitur berikut
- server lokal 9000
- penggunaan framework Express.Js
- sistem CRUD sederhana seperti :<br>
  a. Menambahkan buku<br>
  b. Melihat buku dari ID dan seluruhnya<br>
  c. Mengedit buku<br>
  d. Menghapus buku<br>
- melakukan test API dengan method POST, GET, PUT, DELETE di Postman

# Folder Structure
```txt
bookshelf-API/
├── src/
    ├── books.js       # Menampung objek books berupa Array
    ├── controller.js  # Logic CRUD
    ├── routes.js      # Rute Method GET, POST, PUT, DELETE
    └── server.js      # Eksekusi server dengan port 9000
├── package-lock.json
└── package.json
```
